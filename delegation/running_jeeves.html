Compiling the Jeeves Interpreter

Software

You will need the following to compile and run the Jeeves software.

  • OCaml Version 3.11.0 or above.
  • GCC. Any standard version should be fine.
  • The Yices SMT Solver. You will want to install a dynamically linked version for the Yices C API to work properly.
